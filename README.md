# RapidKeys
A web app designed to test/improve one's typing speed and accuracy, built with Angular and deployed on Firebase.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

<h3>Functionality</h3>

> WPM (Words Per Minute) Calculation
> 
> Error Logging
> 
> Accuracy Calculation
>
